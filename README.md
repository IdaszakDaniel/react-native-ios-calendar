# react-native-ios-calendar

React Native iOS calendar

## Installation

```sh
npm install react-native-ios-calendar
```

## Usage

```js
import IosCalendar from "react-native-ios-calendar";

// ...

const result = await IosCalendar.multiply(3, 7);
```
